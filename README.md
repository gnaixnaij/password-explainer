# Password Explainer

[![Ko-fi](https://img.shields.io/badge/Sponsor-Ko--fi-FF5E5B?style=flat-square&logo=ko-fi)](https://ko-fi.com/gnaixnaij)

See exactly how your password would be cracked. Type a password, get real attack timelines for online, offline, and botnet cracking.

👉 **https://gnaixnaij.github.io/password-explainer/**

## Features

- Real-time strength analysis with entropy calculation
- Attack time estimates for 3 threat levels (online, offline GPU, botnet)
- Character set and entropy breakdown
- Detects common/weak passwords
- Privacy-first: everything runs in your browser, nothing is sent to a server

## Local dev

```bash
git clone git@github.com:gnaixnaij/password-explainer.git
cd password-explainer
# Open index.html in a browser — no build step needed
```
